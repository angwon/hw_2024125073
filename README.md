# 📍 세이프맵 (SafeMap) 
> **AI 기반 동네 안전 지도 및 안전 경로 추천 서비스**

사용자 신고 데이터를 기반으로 위험 정보를 수집하고, AI 분석을 통해 실시간 안전 지도와 최적의 안전 경로를 제공하는 스마트 안전 서비스입니다.

---

## 1. 프로젝트 개요
* **개발 기간:** 2026.03.16 ~ 2026.06.16 (3개월)
* **담당자:** * **안지원 (PM):** 요구사항 총괄, 일정 관리
  * **김파랑:** 시스템 설계, AI 알고리즘 정의
  * **김하양:** 웹/앱 프론트엔드 개발, UI/UX
  * **김초록:** DB 설계 및 데이터 관리

## 2. 핵심 기능
1. **위험 요소 신고:** GPS 기반 자동 위치 등록 및 사진 첨부
2. **위험 지도 자동 생성:** 신고 데이터를 시각화한 동네 위험지도
3. **AI 기반 위험도 분석:** 신고 빈도, 유형 등을 통한 객관적 위험 수준 제공
4. **안전 경로 추천:** 위험 지역을 회피하는 보행 경로 안내
5. **실시간 위험 알림:** 위험 지역 접근 시 푸시 알림
