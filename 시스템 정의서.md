## 4. 핵심 기능
### 기능 1) 위험 요소 신고 기능
* 사용자가 동네에서 발견한 위험 요소를 직접 신고할 수 있는 기능
* GPS를 활용한 자동 위치 등록, 사진 첨부 가능, 위험 유형 선택 가능

### 기능 2) 위험 지도 자동 생성 기능
* 사용자들이 신고한 위험 정보를 기반으로 동네 위험지도를 자동 생성하는 기능
* 지도 구성: 위험 위치 마커 표시, 위험 유형별 색상 구분, 위험도 단계 표시

### 기능 3) AI 기반 위험도 분석 기능
* 시스템에 등록된 신고 데이터를 분석하여 위험도를 자동으로 판단하는 기능
* **분석 기준:** 동일 위치 신고 횟수, 신고 유형, 최근 신고 시간, 사용자 평가
* 객관적인 위험 수준 제공

### 기능 4) 안전 경로 추천 기능
* 사용자가 목적지를 설정하면 위험 지역을 피해 이동할 수 있는 안전 경로를 추천하는 기능
* 일반 경로와 안전 경로의 비교 기능 제공

### 기능 5) 실시간 위험 알림 기능
* 사용자가 위험 지역 근처에 접근하면 스마트폰 알림을 통해 위험 정보를 제공하는 기능
* 위험 상황을 사전에 인지하고 예방 가능

## 5. 유사 관련 시스템
### **Citizen**
* 지역 주민들이 범죄, 사고, 위험 상황 등을 신고하고 공유하는 커뮤니티 안전 앱
* 미국에서 많이 사용되는 지역 안전 플랫폼, 사용자 신고 기반 정보 공유
* **주요 기능:** 범죄 및 사고 신고, 주변 안전 정보 확인, 실시간 알림
* **공통점:** 사용자 참여 기반 위험 정보 수집, 지역 안전 정보 제공
