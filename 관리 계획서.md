# 📍 동네 위험지도 자동 생성 및 안전 경로 추천 서비스

> **AI 분석을 통한 스마트 안전 네비게이션 및 실시간 알림 서비스**
> 사용자 신고 데이터를 기반으로 동네의 위험 요소를 시각화하고, 최적의 안전 경로를 제공하여 더 안전한 지역 사회를 만듭니다.

---

## 📝 프로젝트 개요 (Project Overview)

본 프로젝트는 지역 내 위험 정보를 수집하고 AI 분석을 통해 **동네 안전 지도**를 생성합니다. 단순한 정보 제공을 넘어, 목적지 설정 시 위험 지역을 회피하는 **최적의 안전 경로**를 추천하고 실시간 알림을 제공하는 스마트 안전 서비스 개발을 목표로 합니다.

## ✨ 주요 기능 (Key Features)

| ID | 기능명 | 상세 설명 |
| :-- | :--- | :--- |
| **F01** | **위험 요소 신고** | GPS를 활용하여 위치, 사진, 위험 유형을 사용자가 직접 신고 |
| **F02** | **위험 지도 자동 생성** | 신고된 정보를 바탕으로 마커 표시 및 위험도별 색상 구분 지도 생성 |
| **F03** | **AI 기반 위험도 분석** | 신고 횟수, 유형, 최근 신고 시간 등을 분석하여 객관적 위험 수치 산출 |
| **F04** | **안전 경로 추천** | 목적지 설정 시 위험 지역을 회피하는 최적의 안전 이동 경로 제공 |
| **F05** | **실시간 위험 알림** | 위험 지역 접근 시 스마트폰 푸시 알림을 통해 사전 주의 정보 제공 |

## 🛠 기술 스택 및 개발 환경 (Tech Stack & Environment)

* **개발 모델:** 나선형 모델 (Spiral Model) - 점진적 개발 및 위험 최소화
* **설계 방법론:** 컴포넌트 기반 설계 (CBD) - 독립적이고 교체 가능한 부품 단위 개발
* **주요 기술:**
    * GPS 기반 위치 서비스
    * AI 데이터 분석 및 위험도 산출 알고리즘
    * 실시간 푸시 알림 시스템

## 📚 용어 정의 (Glossary)

* **GPS:** 인공위성을 이용해 현재 본인의 위치를 파악하는 기술
* **나선형 모델:** 소프트웨어 개발 과정이 반복적이고 점진적으로 진행되는 모델
* **컴포넌트 기반 설계(CBD):** 시스템을 독립적인 부품 단위로 나누어 개발하고 조립하는 방법론
