## [cite_start]1.2 프로젝트 개요 [cite: 19]

### [cite_start]1.2.1 프로젝트 정의 [cite: 20]
[cite_start]사용자 신고 데이터를 기반으로 지역 내 위험 정보를 수집하고 **AI 분석**을 통해 동네 안전 지도를 생성하여 **최적의 안전 경로 추천 및 실시간 알림**을 제공하는 스마트 안전 서비스를 개발합니다. [cite: 21]

### [cite_start]1.2.2 주요 기능 설명 [cite: 22]

| ID | 기능명 | 설명 |
|:---:|:---:|:---|
| **F01** | **위험 요소 신고** | [cite_start]사용자가 GPS를 활용해 위치, 사진, 위험 유형을 직접 신고  |
| **F02** | **위험 지도 자동 생성** | [cite_start]신고된 정보를 바탕으로 마커 표시 및 위험도별 색상 구분 지도 생성  |
| **F03** | **AI 기반 위험도 분석** | [cite_start]신고 횟수, 유형, 최근 신고 시간 등을 분석하여 객관적 위험 수치 산출  |
| **F04** | **안전 경로 추천** | [cite_start]목적지 설정 시 위험 지역을 회피하는 최적의 안전 이동 경로 제공  |
| **F05** | **실시간 위험 알림** | [cite_start]위험 지역 접근 시 스마트폰 푸시 알림을 통해 사전 주의 정보 제공  |
