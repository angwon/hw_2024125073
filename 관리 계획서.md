## 1. 서론

### 1.1 문서의 목적 및 범위
본 프로젝트 계획서는 **동네 위험지도 자동 생성 및 안전 경로 추천과 실시간 알림 서비스**에 대한 전반적인 내용을 정리하여 프로젝트를 진행하는 데 있어 각 세부 사항의 지표로 삼는 것을 목적으로 한다.

### 1.2 프로젝트 개요
#### 1.2.1 프로젝트 정의
사용자 신고 데이터를 기반으로 지역 내 위험 정보를 수집하고 AI 분석을 통해 동네 안전 지도를 생성하여 최적의 안전 경로 추천 및 실시간 알림을 제공하는 스마트 안전 서비스를 개발한다.

#### 1.2.2 주요 기능 설명 (ID 기반)
| ID | 기능명 | 설명 |
| :--- | :--- | :--- |
| **F01** | **위험 요소 신고** | 사용자가 GPS를 활용해 위치, 사진, 위험 유형을 직접 신고 |
| **F02** | **위험 지도 자동 생성** | 신고된 정보를 바탕으로 마커 표시 및 위험도별 색상 구분 지도 생성 |
| **F03** | **AI 기반 위험도 분석** | 신고 횟수, 유형, 최근 신고 시간 등을 분석하여 객관적 위험 수치 산출 |
| **F04** | **안전 경로 추천** | 목적지 설정 시 위험 지역을 회피하는 최적의 안전 이동 경로 제공 |
| **F05** | **실시간 위험 알림** | 위험 지역 접근 시 스마트폰 푸시 알림을 통해 사전 주의 정보 제공 |

## 5. 기타 제약 사항 및 리스크 관리

### 5.1 기타 제약 사항
* **GPS 오차:** 실내/고층 빌딩 밀집 구역의 오차를 고려한 보정 기능 필요.
* **법적 준수:** 개인정보 보호법 및 위치정보법에 따른 사용자 동의 절차 필수.
* **데이터 신뢰성:** 악의적 허위 신고 방지를 위한 필터링 알고리즘 적용.
